
# Russian Phonetic Keyboard Layout

This project provides a Russian Phonetic keyboard layout for macOS. The layout closely resembles the Windows ЯВЕРТЫ layout. After years of using this layout and struggling to find it again after reinstalling my Mac, I decided to make it available online for everyone.

## Usage


You can download the latest release from the [Releases page](https://github.com/yourusername/russian-phonetic/releases).


Each release includes:
- Source code (`.zip`, `.tar.gz`)
- Pre-built RussianPhonetic.dmg


### Installation
1. Drag and drop the `RussianPhonetic.bundle` into the `Keyboard Layouts` folder (usually located at `/Library/Keyboard Layouts/`).
2. Log out and log back in to refresh the available keyboard layouts.
3. Go to System Preferences → Keyboard → Input Sources and add the "Russian - ЯВЕРТЫ" layout.
